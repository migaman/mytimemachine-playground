# mytimemachien-playground

Playground for mytimemachine contract

## Steps (Windows)

 Start Ganache GUI
    
    truffle.cmd migrate --reset
    
Start webapp
    
    npm start
    
Open Browser 
    
    http://localhost:9001/


Open MetaMask with Mnemonic from Ganache
    


### Nonce error

Nonce error in MetaMask during sending. -> Reset Account in MetaMask Settings.
